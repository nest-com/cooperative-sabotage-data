This repository includes matlab scripts and result data that are analysed and presented in the following paper that is currently under review in a scientific journal: 

"Cooperative Sabotage: The Synergistic Threat of Industroyer and MaDIoT in a Power System"
Authors: Nestor Rodriguez-Perez, Juan Lozano, Lukas Sigrist, Gregorio Lopez, Álvaro Cardenas

Description of folders:

* "hiddenSignal_PF" includes the data regarding the simulation of the hidden signal in PowerFactory together with a load curve. It also includes the python script used to detect the hidden signal using FastICA
* "results_PF" includes the result data of the simulation of the attacks in PowerFactory.
* "signalingPhase_matlab" includes the matlab scripts and data used to support the results in section "Signaling Phase" of the article.
